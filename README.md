# Book-Analysis-SQL
A comprehensive SQL project built using PostgreSQL to analyze a bookstore database. This project focuses on solving real-world business problems by writing basic, intermediate, and advanced SQL queries to extract meaningful insights from sales, customers, books, and inventory data.


## 🚀 Project Overview

This project demonstrates how SQL can be used to analyze bookstore operations and answer business-related questions such as:

- Book sales analysis
- Customer purchase behavior
- Revenue analysis
- Genre-wise sales
- Author performance
- Inventory (stock) management
- Order analysis
